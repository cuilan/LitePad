# LitePad

## 项目概述

LitePad 是一个轻量级代码编辑器，具有以下特点：

- **轻量化**: 追求极致的性能和轻量级设计
- **跨平台**: 支持 Windows、macOS、Linux 等多平台
- **插件化**: 通过插件系统实现功能扩展
- **高度可配置**: 支持丰富的配置选项
